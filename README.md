# Product Microservice

This MS was generate with NestJS

## Installation

1. Clone Respository

2. Install dependencies

```
npm install
```

3. Create file `.env` with `env.template`

4. Run migrations from Prisma

```
npx prisma migrate dev
```

5. Run App

```
npm run start:dev
```
